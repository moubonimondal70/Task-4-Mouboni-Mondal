# E-Commerce Sales Data Visualization

This project analyzes an e-commerce sales dataset using Python, Pandas, Matplotlib, and Seaborn. Various visualizations were created to explore sales performance, customer behavior, payment preferences, marketing effectiveness, and revenue trends.

## Key Analysis

* Revenue by Product
* Revenue by Payment Method
* Revenue by Referral Source
* Order Status Distribution
* Monthly Revenue Trend
* Quantity vs Total Price Analysis
* Coupon Performance Analysis
* Correlation Heatmap

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab / Jupyter Notebook

## Objective

To practice data visualization and exploratory data analysis (EDA) by transforming raw sales data into meaningful business insights that support data-driven decision-making.
